# AURASZN Momentum OS v3

## Your Links
After deploying to Vercel, you'll have:

- **`yoursite.vercel.app/guide.html`** — The launch guide (send this to your people)
- **`yoursite.vercel.app`** — The app itself

## How to Share With Your Students

1. Deploy to Vercel (see below)
2. Send your people this link: `yoursite.vercel.app/guide.html`
3. They read the guide, hit ACTIVATE, and they're in the app
4. Each person's data saves privately on their own device

## Deploy to Vercel (Free)

1. Create a GitHub repo called `auraszn-momentum-os`
2. Upload ALL these files to the repo (keep the folder structure)
3. Go to vercel.com → Sign up with GitHub
4. Click "Import Project" → Select your repo
5. Framework: Vite (auto-detected)
6. Click Deploy
7. Your site is live in ~60 seconds

## Updating
Push changes to GitHub → Vercel auto-deploys in seconds.
